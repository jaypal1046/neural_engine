Question: How do I debug a Python error?

Ideal Answer: Debugging Python errors step by step:

1. **Read the full traceback** — Python shows exactly which line failed and what kind of error. Read from bottom up.
2. **Identify the error type:**
   - `NameError` → variable not defined or misspelled
   - `TypeError` → wrong argument type (`int` where `str` expected)
   - `IndexError` → list index out of range
   - `KeyError` → key not in dictionary
   - `AttributeError` → calling method on wrong object type
3. **Print intermediate values** — add `print(variable)` before the crash line
4. **Use pdb:** `import pdb; pdb.set_trace()` — step through code line by line
5. **Isolate the problem** — comment out half the code, narrow down the cause
6. **Google the exact error message** — almost every Python error has a StackOverflow answer

**Golden rule:** Never assume. Print the actual value to confirm what you think is true.
