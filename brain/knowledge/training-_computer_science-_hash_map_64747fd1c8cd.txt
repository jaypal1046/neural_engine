Question: What is a hash map?

Ideal Answer: A hash map refers to a data structure that stores key-value pairs with O(1) average-case lookup, insertion, and deletion.

1. Hash function: converts a key (string, int) into an array index.
2. Bucket: the array slot where the value is stored.
3. Collision handling: when two keys hash to the same index — resolved by chaining (linked list per bucket) or open addressing.
4. Load factor: ratio of items to buckets. When too high (> 0.75), the map resizes.

Python dict, Java HashMap, and C++ unordered_map are all hash maps. Used everywhere: caches, frequency counts, database indexes, symbol tables.
