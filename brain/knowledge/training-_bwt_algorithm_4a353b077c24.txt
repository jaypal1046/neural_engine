Question: What is BWT?

Ideal Answer: BWT (Burrows-Wheeler Transform) is a reversible text transformation that rearranges bytes to make similar bytes cluster together — making compression much easier.

Example with 'BANANA':
- Generate all rotations, sort them alphabetically
- Take the last column → 'ANNBAA'
- Similar letters are now adjacent → run-length encoding and Huffman coding work much better

After BWT we apply Move-To-Front (MTF) coding and RLE to cluster zeros, then ANS entropy coding. The result for text: 55–70% compression ratios.
