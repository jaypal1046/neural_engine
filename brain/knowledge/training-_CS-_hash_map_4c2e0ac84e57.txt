Question: What is a hash map?

Ideal Answer: A hash map (also called hash table or dictionary) is a data structure mapping keys to values using a hash function.

**How it works:**
1. You provide a key (e.g., 'apple')
2. A hash function converts it to an integer (array index)
3. Value is stored at that index
4. Lookup: hash the key again → retrieve value directly

**Time complexity:** O(1) average for get/set/delete (much faster than O(n) linear search)

**Python example:**
```python
frequency = {}
for char in 'banana':
    frequency[char] = frequency.get(char, 0) + 1
# frequency = {'b':1, 'a':3, 'n':2}
```

**Collisions:** Two keys hash to same index. Resolved by chaining (linked list at each slot) or open addressing.

**Use cases:** Frequency count, caching, sets, database indexes.
