Question: What is the difference between a stack and a queue?

Ideal Answer: Both are linear data structures, but with opposite ordering:

**Stack (LIFO — Last In, First Out):**
- Like a stack of plates — last one placed is first removed
- Operations: `push` (add to top), `pop` (remove from top)
- Use case: undo/redo, call stack, DFS graph traversal

**Queue (FIFO — First In, First Out):**
- Like a checkout line — first one in is first served
- Operations: `enqueue` (add to back), `dequeue` (remove from front)
- Use case: BFS graph traversal, print queue, request handling

**Memory trick:** Stack = cafeteria plates. Queue = airport security line.
