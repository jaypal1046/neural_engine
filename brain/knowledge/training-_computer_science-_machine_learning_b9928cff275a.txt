Question: What is machine learning?

Ideal Answer: Machine learning refers to the field of AI where systems learn patterns from data without being explicitly programmed.

1. Supervised learning: the model trains on labeled examples (input → correct output) to make predictions.
2. Unsupervised learning: the model finds patterns in unlabeled data (e.g., clustering).
3. Reinforcement learning: the model learns by taking actions and receiving rewards or penalties.
4. Deep learning: uses multi-layer neural networks to learn complex representations automatically.

Key applications: image recognition, speech recognition, recommendation systems, natural language processing. Popular frameworks: TensorFlow, PyTorch, scikit-learn.

Skills demonstrated: types, supervised vs unsupervised, applications, frameworks
