Question: What is the difference between a list and a tuple in Python?

Ideal Answer: Both are ordered sequences in Python, but with a key difference:

**List** - mutable (can be changed after creation):
```
fruits = ['apple', 'banana', 'cherry']
fruits.append('date')     # OK
fruits[0] = 'avocado'     # OK
```

**Tuple** - immutable (cannot be changed after creation):
```
point = (3, 4)
point[0] = 5  # TypeError: cannot modify
```

**When to use each:**
- **List:** when you need to add/remove/modify (shopping cart, log entries)
- **Tuple:** for fixed data that shouldn't change (coordinates, RGB colors, database record)
- **Tuple is faster** to iterate and uses less memory
- **Tuple can be a dict key;** list cannot (because lists are mutable)
