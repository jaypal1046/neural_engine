Question: What is binary search?

Ideal Answer: Binary search refers to an efficient algorithm for finding a target value in a SORTED array by repeatedly halving the search range.

1. Start with the full array. Check the middle element.
2. If middle == target: found! Return the index.
3. If middle > target: search the LEFT half.
4. If middle < target: search the RIGHT half.
5. Repeat until found or the range is empty.

Time complexity: O(log n) — each step halves the search space.
Space complexity: O(1) iterative, O(log n) recursive.

Example: searching for 7 in [1,3,5,7,9,11] — check 5 → check 9 → check 7 → found in 3 steps instead of 6.
