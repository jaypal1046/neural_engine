Question: What is machine learning?

Ideal Answer: Machine learning is a subfield of AI where systems learn patterns from data instead of being explicitly programmed.

**3 main types:**
1. **Supervised learning** — learn from labeled examples. Input → output pairs. (email → spam/not-spam)
2. **Unsupervised learning** — find hidden patterns in unlabeled data. (customer segmentation, anomaly detection)
3. **Reinforcement learning** — agent learns by trial and error, maximizing reward. (game AI, robotics)

**Typical pipeline:**
- Collect data → Clean data → Feature engineering → Train model → Evaluate → Deploy

**Popular algorithms:** Linear regression, Decision trees, Random forests, SVM, Neural networks, k-means clustering.
