Question: What is the difference between == and is in Python?

Ideal Answer: `==` checks **value equality** — do the two objects have the same value?
`is` checks **identity** — are the two objects the SAME object in memory?

```python
a = [1, 2, 3]
b = [1, 2, 3]
c = a

a == b   # True  ← same value
a is b   # False ← different objects in memory
a is c   # True  ← c points to same object as a
```

**Common pitfall:**
```python
if x == None:   # works but not idiomatic
if x is None:   # CORRECT — None is a singleton; always use 'is'
```

**Rule:** Use `==` for values. Use `is` only for singletons: `None`, `True`, `False`.
