┌──────────────────────────────────────────────────────────────────────────┐
│ USAGE EXAMPLES                                                           │
└──────────────────────────────────────────────────────────────────────────┘

  1. Quantize FP32 model to Q4_0 (8x compression):
     $ neural_engine quantize_model model.bin model.q4_0 --format Q4_0

  2. Quantize FP32 model to Q8_0 (4x compression):
     $ neural_engine quantize_model model.bin model.q8_0 --format Q8_0

  3. Restore quantized model back to FP32:
     $ neural_engine dequantize_model model.q4_0 model_restored.bin

┌──────────────────────────────────────────────────────────────────────────┐
│ TECHNICAL DETAILS                                                        │
└──────────────────────────────────────────────────────────────────────────┘

  Quantization Format: Q4_0
    - Block Size:  32 floats per block
    - Block Struct:
        uint16_t scale;   // FP16 scale factor (2 bytes)
        uint8_t qs[16];   // 32 × 4-bit values (16 bytes)
    - Total:       18 bytes per block (vs 128 bytes FP32)
    - Ratio:       7.11x compression

  Quantization Format: Q8_0
    - Block Size:  32 floats per block
    - Block Struct:
        uint16_t scale;   // FP16 scale factor (2 bytes)
        int8_t qs[32];    // 32 × 8-bit values (32 bytes)
    - Total:       34 bytes per block (vs 128 bytes FP32)
    - Ratio:       3.76x compression

  File Format (.q4_0 / .q8_0):
    [8 bytes]  num_blocks (size_t)
    [N bytes]  blocks (num_blocks × sizeof(Block))
