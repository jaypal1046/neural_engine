### Embeddings (Word2Vec)
```bash
# Train on your corpus
bin\neural_engine.exe embed_train corpus.txt data\embeddings.bin --epochs 3

# Find similar words
bin\neural_engine.exe embed_similar "compression"

# Word analogies (king:man::queen:woman)
bin\neural_engine.exe embed_analogy corpus.txt king man woman
```

### RAG (Question Answering)
```bash
# Add knowledge
bin\neural_engine.exe rag_add_doc mybook.txt

# Load embeddings for semantic search
bin\neural_engine.exe rag_load_embeddings data\embeddings.bin

# Ask questions
bin\neural_engine.exe rag_ask "What is compression?"
```

### Memory (Learning from Feedback)
```bash
# Record conversation
bin\neural_engine.exe memory_record "What is BWT?" "BWT is..." 0.9

# Give feedback
bin\neural_engine.exe memory_feedback positive

# Correct mistakes (AI learns!)
bin\neural_engine.exe memory_correct "Actually, BWT uses O(n log^2 n)"

# Export/import memory
bin\neural_engine.exe memory_export brain.bin
bin\neural_engine.exe memory_import brain.bin
```
